EngMyanDictionary
=================

English-Myanmar Dictionary

## Change Log

#### Version 3.1.0
- Improve UI
- Apply Lollipop Theme
- Bug fix

#### Version 3.0.0
- Improved search result.
- Add wildcards search. Eg: *ist, th?n.
- Add Clear Recents button on recents list.
- Add Edit Favorites button on favorites list.

- Bug fix.
- Add Favorites.
